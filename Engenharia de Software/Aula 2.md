# Desenvolvimento de Software
Pensamento no desenvolvimento de um software.
O foco está em todos os aspectos da produção de um software e considerar todo o contexto em que o sistema está inserido. Pensar nos estágios de inicio até a sua manutenção, além da sua escalabilidade. 

## Quem determina os requisitos mínimos de um sistema
Quando uma equipe escolhe programas específicos para desenvolver o software (como SGBD, linguagem de programação, etc.), essa escolha é levada em consideração porque determinados sistemas precisam de hardwares específicos 
## Processo de software
Abordagem sistemática. Uma sequência de atividades que leva à produção de um sistema.
## Detalhes e extras
Obs: Não existe aplicação em tempo real. A informação nunca é transmitida de forma instantânea.
